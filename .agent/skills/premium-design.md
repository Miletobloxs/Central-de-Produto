# Skill: Premium Design Excellence

This skill enforces the "Anti-Safe Harbor" design philosophy for the dashboard.

## Mandates

- **Layout Diversity**: Avoid generic 12-column grids; use asymmetrical balances where appropriate.
- **Visual Depth**: Implement glassmorphism, subtle gradients, and depth layers (z-index strategy).
- **Micro-interactions**: Every interactive element must have a defined hover/active state and subtle transition.
- **Auditing**: Use `ui-visual-validator` to ensure implementation matches the `DESIGN.md` intentions.

## Triggers

- "Melhorar estética"
- "Auditoria de UI"
- "Implementar novo componente premium"
