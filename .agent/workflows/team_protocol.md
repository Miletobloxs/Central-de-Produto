# Protocolo de Orquestração Global de Agentes (PGOA) 🎼

Este projeto opera sob o regime de **Orquestração de Elite**. Nenhuma ação é realizada sem a consulta aos especialistas e ferramentas adequadas.

## 🎼 O Orquestrador (Antigravity)
O> [!IMPORTANT]
> **Este protocolo é perpétuo para este projeto.** Ele garante que cada entrega seja construída não por uma IA genérica, mas por um time de elite orquestrado para a perfeição em qualquer domínio.
> **Fluxo de Teste**: Ao finalizar uma tarefa (TDD Green), o agente deve fornecer o comando `docker-compose up --build -d` para o usuário disparar o build manual antes do teste. 🚀💎✨🛡️

## 👥 Especialistas Prontos para Ação
*   **📐 Arquiteto**: Planejamento de infraestrutura e lógica.
*   **🎨 Designer**: Estética premium e UX impecável (UI/UX Pro Max).
*   **⚙️ Desenvolvedor (FE/BE)**: Implementação de código limpo e performático.
*   **🔒 Auditor**: Segurança, RLS e proteção de dados.
*   **🚀 DevOps**: CI/CD, Tags de versão e Estabilidade.
*   **🕵️ QA**: Verificação rigorosa e testes de interface.

## 🛠️ Skills & Ferramentas
Utilize sempre:
1.  `UI/UX Pro Max` para design.
2.  `App Builder` para estruturas.
3.  `Browser Subagent` para validação visual.

## 🔄 Fluxo de Trabalho
1. **Analise**: Qual especialista é o dono deste domínio?
2. **Planeje**: O Arquiteto aprovou a abordagem?
3. **Execute**: O Designer e o Auditor validaram o código?
4. **Verifique**: O QA confirmou o resultado final?
